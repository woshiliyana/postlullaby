# PostLullaby Photo Beat Spike

A browser-only prototype for testing whether one photo reacting to one song creates a compelling memorial-video moment.

**Original Spark** is the 15-second test scene. **Your Song** plays the selected local track in full in the current browser; this spike does not export a video.

## Run locally

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Media and copyright boundary

- **Original Spark** is a repository-owned sample created for this prototype.
- A song selected from your device stays in your browser and is never uploaded.
- You must hold the rights to any song you select.
- This spike does not provide AI restyling, server upload, or export.
